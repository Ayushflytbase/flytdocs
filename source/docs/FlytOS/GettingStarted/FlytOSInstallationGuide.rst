Setup FlytOS
============


FlytOS is compatible with :ref:`FlytPOD<about_flytpod>`, :ref:`FlytPOD PRO<about_flytpodpro>`, `ODROID-XU4 <http://www.hardkernel.com/main/products/prdt_info.php?g_code=G143452239825>`_, `Raspberry Pi 3 <https://www.raspberrypi.org/products/raspberry-pi-3-model-b>`_, `NVIDIA Jetson TX1 <https://developer.nvidia.com/embedded-computing>`_ , `Intel Edison <https://software.intel.com/en-us/iot/hardware/edison>`_ and `Intel Aero Compute Board <https://software.intel.com/en-us/aero/dev-kit>`_. We will soon introduce compatibility with many popular SBCs available in the drone ecosystem. Select your companion computer from the links below for tutorial to install FlytOS:

* :ref:`FlytPOD/PRO<flytpod_guide>`
* :ref:`Odroid XU4<odroid_guide>`
* :ref:`Raspberry Pi 3<rpi_guide>`
* :ref:`Nvidia TX1<tx1_guide>`  
* :ref:`Intel Edison<edison_guide>`
* :ref:`Intel Aero Compute Board<aero_guide>`
